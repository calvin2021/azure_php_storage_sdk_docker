# azuer_php_sdk_docker

this is based on azuer storage sdk. upload file to azure blob.

# step:
####1.change blob user and key in docker-composer.yml
####2.run command
```
docker-compose up -d
```
####3.login docker
####4.run command
```
composer install
```
####5.exec the program
```
php sample/lytech.php
```
